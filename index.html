@novemberizing/storage

"Novemberizing storage" is a library made in JavaScript that allows you to access storage and easily perform tasks such as creating, deleting, and modifying resources.

@novemberizing/app

"Novemberizing app" is a framework that defines the basic skeleton of a server so that a server application can be created simply by implementing a service or module or defining it in configuration.
